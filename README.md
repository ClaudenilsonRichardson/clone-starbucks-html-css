# ☕ Clone do Starbucks

## 📌 Sobre o projeto
Este projeto foi desenvolvido como parte dos meus primeiros estudos em desenvolvimento web, utilizando HTML e CSS.

O objetivo foi recriar a interface de uma página do Starbucks, praticando estruturação de layout, estilização e organização de arquivos.

---

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3

---

## 🎯 Funcionalidades
- Estrutura de página web completa
- Estilização com CSS
- Organização de imagens e layout

---

- Uso de HTML semântico
- Estilização com CSS
- Organização de arquivos em projetos web
